Get help from the community

StackOverflow has a great and active community focused around W3C Standards and W3C developer tools. To ask a question to the W3C Community, we invite you to use the tag w3c, eventually followed by one of these:

w3c-validation

css-validator

html-validation

Get involved in great projects

Developers are welcome to get involved by contributing code, either to existing tools or proposed future projects.

But it's not the only way to get involved in making W3C developer tools better. Testing, bug reports, suggestions, or help in creating good documentation are equally important! Most projects will have a Feedback page, and you can report bugs, send test cases and patches.

You will find more information to get started in each project on their own web page.
